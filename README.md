# Todo App

Todo App is a typical todo web application with added functionality of icon suggestion using tensorflow.js

## Installation

```bash
git clone https://github.com/DeepAgrawal/TodoApp-ML.git
cd TodoApp-ML
npm install
```

## Usage

```bash
npm start
```
